Some of My WordPress Plugins

Get GitHub Code

Get GitHub Code 是一个在 wordpress 调用 GitHub API v2 来获取某个用户项目中的代码片段的插件。该插件通过 Jsonp 获取数据，存储在本地 cache 文件中，可以实时获取并更新。

➥Read More

Screenshot:

Features:

  • PHP 后台 Jsonp 获取数据，保存在 cache 文件中。
  • 在 Wordpress 提供方便的设置接口，可以手动修改 cache 存活时间。
  • 内嵌 ColorCode 代码高亮插件需要用到的勾。
  • 获取最新更新的内容, 可扩展性强。

ColorCode

ColorCode 是一个 WordPress 的代码高亮插件，提供一套色彩的配色方案。主要支持 XML, HTML, CSS 和 Javascript 四种语言的代码高亮。

➥Read More

Screenshots:

Features:

  • 支持 XML, HTML, CSS 和 Javascript 语言的动态高亮显示。
  • 1.0 版本提供 4 套精心设计的配色方案。
  • 可在 WordPress 后台动态配置色彩方案，是否显示行号等。
